#### Pre-Installation:
1. Install NodeJS
2. Install preferred code editor
3. Install git

#### Installation:
1. Clone repo
2. `cd prosemirror-accented-demo`
3. `npm install`
4. `npm run start`
5. [http://localhost:4444](http://localhost:4444)

#### Requirements:
1. npm 8.0.0+
2. node 16.0.0+
3. Visual Studio Code OR Webstorm
